CSS 背景

CSS 背景属性用于定义HTML元素的背景。

CSS 属性定义背景效果:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

你可以看到这个段落是 被设定的 CSS 渲染的。

当使用简写属性时，属性值的顺序为：

  • background-image
  • background-repeat
  • background-attachment 背景图像是否固定或者随着页面的其余部分滚动。
